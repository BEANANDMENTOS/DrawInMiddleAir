# DrawInMiddleAir ✍️

วาดกลางอากาศด้วยการ "จีบนิ้ว" — เว็บไฟล์เดียว (HTML + JS) ตรวจจับมือแบบเรียลไทม์ด้วย
**MediaPipe Hand Landmarker** จีบนิ้วโป้ง+นิ้วชี้เข้าหากัน = ปากกา, ปล่อย = ยกปากกา
ใช้ได้ทั้งคอม, iPad และมือถือ

## ⚠️ ข้อสำคัญ
- ต้องเปิดผ่าน **HTTPS** หรือ **http://localhost** เท่านั้น (กล้องไม่ทำงานบน `file://` หรือ http วงแลน)
- ครั้งแรกต้องต่อ **อินเทอร์เน็ต** เพื่อโหลดโมเดลตรวจจับมือจาก CDN
- คำสั่งเสียงบน iPhone/iPad: เปิดผ่าน Safari ปกติ, กดปุ่มไมค์แล้วอนุญาตไมค์, และตรวจว่า Dictation/Speech Recognition เปิดอยู่ใน Settings

## รันบนเครื่อง (โลคัล)
```bash
python -m http.server 8770
# แล้วเปิด http://localhost:8770
```

## วิธีใช้
- ✋ ยกมือเข้ากล้อง → เห็นโครงมือทับบนภาพ (แปลว่า track ติด)
- 👌 จีบนิ้วโป้งชนนิ้วชี้ = เริ่มวาด · ปล่อย = หยุด
- 🎨 เลือกสี · ปรับขนาดเส้นด้วย slider · 🧽 ยางลบ · 🗑️ ล้างทั้งหมด (หรือกด `c`)
- 🎙️ กดไมค์แล้วพูดไทย/อังกฤษ เช่น `สีแดง`/`red`, `สีเหลือง`/`yellow`, `สีเขียว`/`green`, `สีฟ้า`/`blue`, `สีขาว`/`white`, `ยางลบ`/`eraser`, `ล้าง`/`clear`, `ย้อนกลับ`/`undo`, `เส้นใหญ่`/`bigger`, `เส้นเล็ก`/`smaller`, `size 20`
- 📸 ปุ่มชัตเตอร์ (หรือ `Space`) = ถ่ายเก็บผลงาน (รวมเส้นที่วาด) → ดาวน์โหลด / บันทึกลงคลังภาพ
- 🔄 สลับกล้องหน้า/หลัง (สลับแล้วล้างภาพวาด เพราะพิกัดเปลี่ยน)

## เทคนิค
- ตรวจจับมือ: [MediaPipe Hand Landmarker](https://developers.google.com/edge/mediapipe/solutions/vision/hand_landmarker) — 21 จุด/มือ, GPU, ~30fps
- "จีบ" = ระยะ ปลายนิ้วโป้ง (จุด 4) ↔ ปลายนิ้วชี้ (จุด 8) เทียบขนาดฝ่ามือ แล้วดูว่าต่ำกว่าเกณฑ์
- มี hysteresis กันสั่น: เข้าโหมดวาดเมื่อ ratio < 0.35, ออกเมื่อ > 0.50 (ปรับได้ในไฟล์ `index.html`)
