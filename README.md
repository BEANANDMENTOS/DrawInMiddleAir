# DrawInMiddleAir

เว็บกล้องถ่ายรูปแบบเรียลไทม์ — ไฟล์เดียวจบ (HTML + JavaScript) ใช้ได้ทั้งคอม, iPad และมือถือ

## ⚠️ ข้อสำคัญ
กล้องทำงานเฉพาะบน **HTTPS** หรือ **http://localhost** เท่านั้น
(เปิดผ่าน `file://` หรือ http วงแลน `192.168.x.x` กล้องจะไม่ทำงาน — เป็นข้อกำหนดความปลอดภัยของเบราว์เซอร์)

## รันบนเครื่อง (โลคัล)
```bash
python -m http.server 8770
# แล้วเปิด http://localhost:8770
```

## ฟีเจอร์
- 📹 กล้องเรียลไทม์ — เริ่มที่กล้องหลัง, ปุ่มสลับหน้า/หลัง
- 📸 ถ่ายรูป — ปุ่มชัตเตอร์ หรือกด `Space` บนคอม
- 💾 บันทึกลงเครื่อง — ดาวน์โหลด (คอม/Android) หรือ บันทึก/แชร์ลงคลังภาพ (iPhone/iPad)
- 🪞 กล้องหน้าพลิกภาพให้ตรงกับที่เห็น, เซฟภาพเต็มความละเอียด (JPEG 95%)
- 📱 responsive รองรับทุกขนาดจอ + กันขอบ notch
